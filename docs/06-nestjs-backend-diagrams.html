Current Backend Files

main.ts Starts the app, validation, CORS, Swagger.
AppModule Connects ConfigModule, PrismaModule, AuthModule.
AuthController Defines /auth/register, /auth/login, /auth/me.
AuthService Hashes passwords, creates users, signs JWTs.
DTOs Define API data shapes and validation rules.
JwtStrategy Reads and validates bearer tokens.
Guards Protect routes by authentication and role.
PrismaService Shared database access point.
env.validation.ts Fails startup when required config is missing.
